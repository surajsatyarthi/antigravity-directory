## Bonu's 7-Step Playbook (Free Tools Marketing)

### Step 1: Start with Blank Keyword Search
- Go to Ahrefs Keywords Explorer
- **Leave search blank** and click "Search"
- **Why**: Don't limit yourself to keywords you already know

### Step 2: Apply Include Filters
- **Include keyword**: "AI" (for AI SaaS)
- **Include keyword**: "generator" (for tool-type keywords)
- **Result**: All keywords like "AI [X] generator"

### Step 3: Filter by Keyword Difficulty
- **KD Filter**: <10 (very low competition)
- **Why**: Easy to rank without backlinks

### Step 4: Filter by Search Volume
- **Volume Filter**: >1,000 monthly searches
- **Why**: Worth the effort to create content

### Step 5: Verify Tool Relevance
- **Question**: Will users of this tool be potential customers?
- **Example**: "AI chatbot name generator" ‚Üí Yes (people building chatbots = Site GPT customers)
- **Skip irrelevant keywords** (e.g., "AI image generator" if you sell text tools)

### Step 6: Build Tool in <5 Minutes
- Use **Cursor/AI** to generate tool code
- **Prompt**: "Look at my existing free tools. Create a similar tool for keyword [X]."
- **Result**: Instant tool creation with consistent UI

### Step 7: Publish & Repeat
- Create 50+ free tools over time
- Each tool = separate landing page optimized for keyword
- **Compounding effect**: More tools = more traffic = more leads

## What NOT to Do (Anti-Patterns)

### ‚ùå 1. Creating Thin or Duplicate Content
**Bad**: Auto-generate 100s of MCP pages with identical structure and no unique data
**Result**: Google penalties, deindexing
**Fix**: Add unique data (compatibility, pricing, use cases) to each page

### ‚ùå 2. Over-Reliance on Popular Keywords
**Bad**: Target "AI tools" (KD 89), "MCP server" (KD 67)
**Result**: Never rank, wasted effort
**Fix**: Focus on long-tail (KD <10) like "mcp server for coding automation"

### ‚ùå 3. Ignoring Mobile Optimization
**Bad**: Desktop-only designs
**Result**: 60% of traffic bounces (mobile = majority)
**Fix**: Ensure all 100s of pages are responsive

### ‚ùå 4. Neglecting Data Accuracy
**Bad**: Use scraped GitHub data without verification
**Result**: Legal issues, trust erosion
**Fix**: Only display claimed/verified MCPs (already our model!)

### ‚ùå 5. Spamming Links or Keywords
**Bad**: Buy 1,000 low-quality backlinks, stuff keywords
**Result**: Algorithmic penalties
**Fix**: Natural link building (free tools, vendor links, guest posts)

### ‚ùå 6. Skipping Metrics Tracking
**Bad**: Implement SEO without Google Search Console
**Result**: Blind scaling, wasted resources
**Fix**: Set up GSC day 1, review weekly

### ‚ùå 7. Over-Automating Without Review
**Bad**: AI generates 500 pages with no human QA
**Result**: Generic, low-value content
**Fix**: Manual review for top 20% of pages, spot-check rest

### ‚ùå 8. Focusing on Quantity Over Quality
**Bad**: Publish 100s of pages with no CTAs, no depth
**Result**: High bounce rate, low conversions
**Fix**: Every page must have unique value + clear CTA

### ‚ùå 9. Delaying Updates
**Bad**: Publish once, never update
**Result**: Stale content loses rankings
**Fix**: Quarterly content refreshes for top 50 pages

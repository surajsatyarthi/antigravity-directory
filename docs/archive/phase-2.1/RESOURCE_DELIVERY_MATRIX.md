## DETAILED BREAKDOWN

### 1. MCP Server ($49-149)

**What User Buys**:
A plugin that extends Claude/Antigravity's capabilities. MCPs add tools, integrations, or features.

**What They Get**:
```
1. Installation File
   - GitHub repository link / ZIP download
   - Or NPM package (`npm install @creator/mcp-name`)

2. How to Use
   - Step-by-step installation guide (Markdown)
   - Configuration instructions
   - Troubleshooting guide

3. Documentation
   - API documentation (what the MCP does)
   - Function reference
   - Parameter explanations

4. Examples
   - Code examples showing usage
   - Before/after demonstrations
   - Common use cases

5. Support
   - Creator email for questions
   - GitHub issues (optional)
   - Response time (creator defines)
```

**Format**:
- Immediate: Email with download link + setup guide
- Access: Download expires after 3 days (user keeps files forever)
- Ongoing: Updates from creator (if they push new versions)

**User Can**:
✅ Use on unlimited projects
✅ Share internally (within same organization)
✅ Modify for own use
✅ Update when creator pushes improvements

**User Cannot**:
❌ Redistribute/resell
❌ Claim as own work
❌ Use commercially (unless license says otherwise)

**Refund Policy**: 30 days if non-functional or significantly different from description

### 2. Antigravity Rule ($29-79)

**What User Buys**:
A configuration file that defines custom AI behavior rules for Antigravity IDE.

**What They Get**:
```
1. Rule File (.antigravity-rules or .rules)
   - Plain text file
   - Copy-paste into Antigravity IDE
   - Immediately takes effect

2. Documentation
   - What the rule does (plain language)
   - Why it was created (use case)
   - What behavior changes to expect

3. Examples
   - Before rule: AI behaves one way
   - After rule: AI behaves differently
   - Specific examples of improved output

4. Implementation Guide
   - Where to place the file
   - How to enable/disable
   - How to customize

5. Support
   - Creator's email
   - Slack/Discord (optional)
   - Bug fixes (if rule has issues)
```

**Format**:
- Immediate: Email with file attachment + guide PDF
- Access: User downloads file, keeps it forever
- Read-only: Text file is human-readable (user can see/understand what rule does)

**User Can**:
✅ Copy rule into any project
✅ Modify rule for own needs
✅ Use across unlimited projects
✅ Share within team (check license)

**User Cannot**:
❌ Redistribute as their own rule
❌ Sell the rule
❌ Remove creator attribution

**Refund Policy**: 30 days if rule doesn't produce advertised behavior

## WHAT HAPPENS IF USER REFUNDS?

1. **User loses access** to files (but keeps already downloaded copy - can't prevent)
2. **Creator keeps 0%** of payment (refunded to user)
3. **Antigravity keeps 0%** of payment (refunded to user)
4. **Trust score** may be impacted if creator disputes refund
5. **User can re-purchase** later (no permanent ban)

## TECHNICAL NOTES FOR CREATORS

### Before Creating MCP:
- Will you provide source code or just binary?
- Will you push updates? How often?
- Support response time?
- License type (MIT, Apache, proprietary)?

### Before Creating Rule:
- Is it standalone or requires other rules?
- Will you test against latest Antigravity version?
- Any compatibility notes?

### Before Creating Workflow:
- Will it stay up-to-date with platform changes?
- Required plugins/dependencies?
- Will you provide video walkthrough?

### Before Creating Prompt:
- Is it for Claude, ChatGPT, Gemini, or all?
- Are variations included?
- Example outputs included?

### Before Creating Skill:
- Is it open-source or proprietary?
- What's the license?
- Installation requirements?
- Support SLA (service level agreement)?

## RISK MITIGATION

### For Users:
- Read carefully: What's included? What's not?
- Check reviews: Did others have issues?
- Try demo: Does free version work?
- Request support: Does creator respond quickly?

### For Creators:
- Be explicit: Exactly what is included?
- Set expectations: Will you update this?
- Provide support: Response time?
- Document everything: Good docs = fewer refunds

### For Antigravity:
- Verify delivery: Did user get files?
- Track refunds: Pattern of complaints?
- Protect creators: Prevent false refunds
- Ensure quality: Lowest-rated creators may be suspended
